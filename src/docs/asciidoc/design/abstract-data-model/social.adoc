|===

==== `LiteracyType`
Enumeration of age groups used for economic activity classification.

[cols="1,3", options="header"]
|===
| Value         | Description
| `BOTH_READING_AND_WRITING` | पढ्न लेख्न जानेको
| `READING_ONLY` | पढ्न मात्र जानेको
| `NONE` | पढ्न लेख्न नजानेका
|===
